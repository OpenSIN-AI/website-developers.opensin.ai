# Neural-Bus API

Die Neural-Bus API ermöglicht die Orchestrierung autonomer Agenten-Flotten über standardisierte Endpunkte.

## Schnellstart

```bash
curl -X POST https://api.opensin.ai/v1/agents \
  -H "Authorization: Bearer $OSIN_KEY" \
  -d '{"model": "neural-bus-v1", "task": "research"}'
```

Weitere Details findest du im [A2A Protocol Guide](/docs/a2a-protocol).
